# android 8.0系统应用图标适配
步骤：  
1.新建一个android项目  
2.新建的android项目AndroidManifest.xml里面android:icon和android:roundIcon引用的是mipmap，手动改成drawable  
3.展开项目的res目录，右键res目录->New->Image Asset,  打开导入图片就可以编辑  
4.编辑好最后会生成到各种分辨率的mipmap目录下，把这些文件重新复制到drawable目录下
